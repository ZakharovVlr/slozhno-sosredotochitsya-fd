# Сложно сосредоточиться
Адаптивный одностраничный сайт с переключением цветовых тем.
https://github.com/ZakharovVlr/slozhno-sosredotochitsya-fd.git

## Технологии
- HTML5 (семантическая разметка)
- CSS3 (Grid Layout, Flexbox, переменные, псевдоэлементы)
- Методология БЭМ
- JavaScript (готовый скрипт переключения тем)

## Функции
- Автоматическое определение системной темы
- Ручное переключение на светлую/тёмную/авто
- Адаптивная галерея с ленивой загрузкой

## Как использовать
1. Клонировать репозиторий
2. Открыть index.html в браузере или через Live Server
